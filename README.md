# My Personal Portfolio

Welcome to my Personal Portfolio website

It is a CLI-themed portfolio webiste. The terminal design of the old days is what I have tried to copy. The Only color in the entire site is `lime`. 

When searching for themes for my portfolio website, I came across the idea of having the website to be text based and be displayed like it was shown in an old CRT display. 

So just like old computers, the website starts with a shell and then proceeds to the website, also rendered in CLI.

## Tech Stack
1. React JS
2. Tailwind CSS
3. React Hotkeys

## Surprises

Press the `?` key to see the help menu and gain access to the space shooter game

Try `CTRL` + `C` and see how the site reacts...

No terminal app is completed without the ability to use `j` and `k` for navigation. Would this website have it?

## Usage of AI

AI generated code was used for the keyboard shortcuts, space shooter game and the cmatrix running behind the homepage. Further modified by myself.

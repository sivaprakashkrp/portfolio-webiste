# My Personal Portfolio

Welcome to my Personal Portfolio website

It is a CLI-themed portfolio webiste. The terminal design of the old days is what I have tried to copy. The Only color in the entire site is `lime`. 

## Tech Stack
1. React JS
2. Tailwind CSS
3. React Hotkeys

## Usage of AI

AI generated code was used for the keyboard shortcuts, space shooter game and the cmatrix running behind the homepage. Further modified by myself.
